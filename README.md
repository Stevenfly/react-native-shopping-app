## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/your_username_/Project-Name.git
    ```
2. Install `npm` packages
    ```sh
    npm install
    ```
    ```sh
    npm install -g expo-cli
    ```
3. Start the development server
    ```sh
    expo start
    ```
